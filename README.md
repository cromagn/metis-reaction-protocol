# Metis BLE Protocol Reverse Engineering

![License: BSD-3](https://img.shields.io/badge/License-BSD--3-blue.svg)

Repository dedicated to the exploration, documentation and reverse engineering of the **BLE protocol used by Metis Reaction Lights**.  
The goal is to make communication open and understandable, enabling integration, custom tooling, automation and experimentation for developers, makers and researchers.
